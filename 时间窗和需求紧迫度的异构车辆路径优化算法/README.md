# Heterogeneous Emergency Vehicle Routing Problem (EVRP-GA)

## 项目名称

异构应急车辆路径优化遗传算法 / Heterogeneous Emergency Vehicle Routing Problem with Genetic Algorithm

## 项目简介

本项目实现了基于遗传算法（Genetic Algorithm）的异构应急车辆路径规划系统，用于解决海岛应急疏散场景下的路径优化问题。

## 问题背景

在应急疏散场景中，需要使用多种异构交通工具（船只和直升机）从多个岛屿疏散人员到安全集散中心。每种交通工具具有不同的特性：
- **船只（Ship）**：容量大（200人）、速度慢（30km/h）、成本低
- **直升机（Helicopter）**：容量小（20人）、速度快（150km/h）、成本高

## 算法特性

- **遗传算法**：使用 DEAP 框架实现的进化算法
- **双染色体编码**：同时优化船只和直升机路径
- **时间窗惩罚**：根据到达时间计算满意度并施加惩罚成本
- **需求紧迫度**：区分严重（severe）、轻度（mild）、安全（safe）三类乘客

## 数据说明

### 岛屿数据
- 10个岛屿 + 1个安全集散中心
- 每个岛屿包含：位置坐标、需求人数（严重/轻度/安全）、时间窗参数（U/T/L）、紧迫度权重

### 车辆参数
| 车辆类型 | 容量 | 速度 | 单位成本 | 固定成本 |
|---------|------|------|---------|---------|
| 船只    | 200  | 30   | 400/h   | 2000    |
| 直升机  | 20   | 150  | 1200/h  | 5000    |

## 算法流程

1. **初始化**：随机生成船只和直升机路径
2. **评估**：计算总成本（固定成本 + 时间成本 + 惩罚成本）
3. **选择**：锦标赛选择
4. **交叉**：双点交叉
5. **变异**：交换、反转、重定位
6. **迭代**：重复上述过程直至满足终止条件
